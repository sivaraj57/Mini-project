Linked List Data Structure

Introduction

A linked list is a linear data structure that consists of a sequence of elements, each of which contains a reference (or link) to the next element in the sequence. Unlike arrays, linked lists do not have a fixed size and can efficiently insert or delete elements at any position within the list.

Structure

A linked list is composed of nodes. Each node has two components:

  • Data: The actual data or value stored in the node.
  • Pointer (or Link): A reference to the next node in the sequence.

The first node is called the "head," and the last node typically points to a null reference, indicating the end of the list.

Types of Linked Lists

There are different types of linked lists, including:

  • Singly Linked List: Each node contains a data element and a reference to the next node.
  • Doubly Linked List: Each node contains a data element, a reference to the next node, and a reference to the previous node, allowing for traversal in both directions.
  • Circular Linked List: A variation of a singly or doubly linked list where the last node points back to the first node, creating a closed loop.

Common Operations

Operations performed on linked lists include:

  • Insertion: Adding a new node at the beginning, end, or any position within the list.
  • Deletion: Removing a node from the list, which involves updating the references of neighboring nodes.
  • Traversal: Iterating through the linked list to access or process each element.
  • Search: Finding a specific element within the linked list.
  • Reversal: Changing the order of elements by reversing the direction of links.

Example

Here's a simple example in Python that demonstrates the use of a singly linked list:


class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

class LinkedList:
    def __init__(self):
        self.head = None

    def append(self, data):
        new_node = Node(data)
        if not self.head:
            self.head = new_node
            return
        current = self.head
        while current.next:
            current = current.next
        current.next = new_node

# Creating a linked list
my_list = LinkedList()
my_list.append(1)
my_list.append(2)
my_list.append(3)

Conclusion

Linked lists are versatile data structures that provide efficient dynamic storage and easy insertion and deletion of elements. They are used in various applications, including implementing dynamic data structures like stacks, queues, and symbol tables.

Understanding linked lists is fundamental for data structure and algorithm study and for solving problems that require dynamic data management.

Doubly Linked List Data Structure

Introduction

A doubly linked list is a linear data structure that extends the concept of a singly linked list by including two references (or links) in each node: one pointing to the next node and another pointing to the previous node. This bidirectional structure allows for traversal in both forward and backward directions.

Structure

A doubly linked list is composed of nodes, and each node has three parts:

  • Data: The actual data or value stored in the node.
  • Next Reference (Link): A reference to the next node in the list.
  • Previous Reference (Link): A reference to the previous node in the list.

Common Operations

Operations performed on doubly linked lists include:

  • Insertion: Adding a new node at the beginning, end, or any position within the list.
  • Deletion: Removing a node from the list, which involves updating the references of neighboring nodes.
  • Forward Traversal: Iterating through the linked list in the forward direction to access or process each element.
  • Backward Traversal: Iterating through the linked list in the backward direction to access or process each element.
  • Search: Finding a specific element within the linked list.
  • Reversal: Changing the order of elements by reversing the direction of links.

Example

Here's a simple example in Python that demonstrates the use of a doubly linked list:


class Node:
    def __init__(self, data):
        self.data = data
        self.next = None
        self.prev = None

class DoublyLinkedList:
    def __init__(self):
        self.head = None

    def append(self, data):
        new_node = Node(data)
        if not self.head:
            self.head = new_node
            return
        current = self.head
        while current.next:
            current = current.next
        current.next = new_node
        new_node.prev = current

Circular Linked List Data Structure

Introduction

A circular linked list is a variation of a singly linked list where the last node points back to the first node, forming a closed loop. In other words, the tail of the list connects to the head, creating a continuous cycle. Circular linked lists are used in various applications where continuous traversal or cycling through elements is needed.

Structure

A circular linked list is composed of nodes, similar to a singly linked list, with each node containing two parts:

  • Data: The actual data or value stored in the node.
  • Next Reference (Link): A reference to the next node in the list. The last node points back to the first node, creating the circular structure.

Common Operations

Operations performed on circular linked lists include:

  • Insertion: Adding a new node at the beginning, end, or any position within the list.
  • Deletion: Removing a node from the list, which involves updating the references of neighboring nodes.
  • Traversal: Iterating through the linked list to access or process each element, which continues indefinitely due to the circular structure.
  • Search: Finding a specific element within the linked list.

Example

Here's a simple example in Python that demonstrates the use of a circular linked list:


class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

class CircularLinkedList:
    def __init__(self):
        self.head = None

    def append(self, data):
        new_node = Node(data)
        if not self.head:
            self.head = new_node
            self.head.next = self.head  # Point back to itself to create the circular structure
            return
        current = self.head
        while current.next != self.head:
            current = current.next
        current.next = new_node
        new_node.next = self.head

# Creating a circular linked list
my_list = CircularLinkedList()
my_list.append(1)
my_list.append(2)
my_list.append(3)

Conclusion

Circular linked lists provide a unique data structure with continuous traversal capabilities. They are used in various applications, including implementing circular buffers, managing tasks in a round-robin scheduling algorithm, and representing cyclical data.

Understanding circular linked lists and their operations is important for data structure and algorithm study and for solving problems that involve cyclic data structures.

Circular Doubly Linked List Data Structure

Introduction

A circular doubly linked list is a linear data structure that combines the features of a doubly linked list and a circular linked list. In this structure, each node contains data, a reference to the next node, and a reference to the previous node. Additionally, the last node connects back to the first node, forming a closed loop with bidirectional traversal capabilities.

Structure

A circular doubly linked list is composed of nodes, and each node has three parts:

  • Data: The actual data or value stored in the node.
  • Next Reference (Link): A reference to the next node in the list.
  • Previous Reference (Link): A reference to the previous node in the list.

The last node points back to the first node, creating the circular structure.

Common Operations

Operations performed on circular doubly linked lists include:

  • Insertion: Adding a new node at the beginning, end, or any position within the list.
  • Deletion: Removing a node from the list, which involves updating the references of neighboring nodes.
  • Forward Traversal: Iterating through the linked list in the forward direction to access or process each element.
  • Backward Traversal: Iterating through the linked list in the backward direction to access or process each element.
  • Search: Finding a specific element within the linked list.

Example

Here's a simple example in Python that demonstrates the use of a circular doubly linked list:


class Node:
    def __init__(self, data):

        self.next = None
        self.prev = None

class CircularDoublyLinkedList:
    def __init__(self):

        new_node = Node(data)
        if not self.head:
            self.head = new_node
            self.head.next = self.head
            self.head.prev = self.head

        while current.next != self.head:

        current.next = new_node
        new_node.prev = current
        new_node.next = self.head
        self.head.prev = new_node
